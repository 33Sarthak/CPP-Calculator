# C++ Calculator

A menu-driven calculator built using C++.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Modulus
- Square

## Concepts Used

- Functions
- While Loops
- Switch Case
- Conditional Statements
- User Input/Output

## How to Run

1. Compile the program
2. Run the executable
3. Choose an operation from the menu
